# JavaScript

[JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) es un lenguaje de programación interpretado [JIT](#jit) con [funciones de primera clase](#funciones-de-primera-clase). Es conocido como el lenguaje script para páginas web. Sin embargo, es utilizado también en ambientes no web, como Node.js. 

La mayoría de navegadores web, como Chrome, Safari, Firefox, Edge, etc. tienen la capacidad de interpretar JavaScript. 

JS es utilizado en la capa de presentación para realizar tareas desde el navegador, con frameworks como React, Angular, Vue.js y otros. También es utilizado en backend con frameworks como Node.js y otros.

## [JIT](https://developer.mozilla.org/en-US/docs/Glossary/Just_In_Time_Compilation)

**Just-In-Time Compilation** es un proceso de compilación en el cual el código es traducido a lenguaje de máquina en tiempo de ejecución (*runtime*), en lugar de ser compilado previo a la ejecución. Esto se logra a partir de lenguajes de alto nivel.

Los compiladores JIT están típicamente analizando el código mientras se ejecuta, e identifican porciones de código que se ejecutan con frecuencia, y si se identifica que habría una ganancia de velocidad de ejecución entonces compila estas fracciones de código a lenguaje de máquina.

## [Funciones de primera clase](https://developer.mozilla.org/en-US/docs/Glossary/First-class_Function#examples)

Se entiende a un lenguaje de programación como un lenguaje con *First-Class functions* cuando las funciones en dicho lenguaje pueden ser tratadas como variables, es decir una función puede ser pasada como argumento de otra función, o una función puede ser el resultado de otra función, o la función puede ser asignada como el valor de una variable. 

## Herramientas a utilizar durante el entrenamiento:

- Navegador web Chrome o Firefox
- [Node.js](https://nodejs.org/en/download/)
- [Visual Studio Code](https://code.visualstudio.com/)
- Extensiones para Visual Studio Code:
    - Quokka: Ayuda a visualizar el resultado de nuestro código en consola.
    - ESLint: Permite aplicar mejores prácticas a nuestro código y formatearlo.
    - Prettier: Permite formatear código de diversos lenguajes. 
    - *JavaScript Debugger* o *Debugger for Firefox*.
    - Better Comments: coloreo de comentarios
    - Live Server: despliega un servidor local para desplegar las aplicaciones directamente en el navegador web.
    - Ajustar el parámetro *Editor › Guides: Bracket Pairs = true*, para visualizar los paréntesis, corchetes y llaves emparejandolos por color.
